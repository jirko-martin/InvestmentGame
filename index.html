Hello :)

This is InvestmentGame - a MaDKit (Java) based implementation of the well known investment game. This game structurally resembles the game theoretic prisoner's dilemma.

The program has originally been written to support Jérôme's master's thesis in psychology where it will be used to perform an experiment on empathy for pain.

By the way : The program's architecture is motivated by a generic proposal on "story-driven-development". Thus the project also serves as a proof-of-concept for the proposed development approach. More information on this issue will hopefully follow soon.

Running the game

You should have Java 7 installed on your computer. To run the game simply download the Zip file, extract it to any folder on your computer's hard drive and double-click on the file InvestmentGame.jar in the main folder.
